- **Python 3.11+**

## Setup

1. **Clone the repository**
2. **Configure Environment**:
   Copy `.env.example` to `.env` and fill in your credentials.
   Required: `BOT_TOKEN`, `OPENROUTER_API_KEY`, `DATABASE_URL`.

3. **Run with Docker**:
   ```bash
   docker-compose up -d
   ```

4. **Run Locally**:
   ```bash
   pip install -r requirements.txt
   alembic upgrade head
   python -m app.bot
   ```
